# Smart Parking System

Final Year Project

## Features
- User registration
- Vehicle management
- Slot reservation
- AI parking detection
- Path guidance

## Tech stack
Frontend: React Native  
Backend: Flask  
Database: MySQL  
AI: Python, OpenCV  

## How to run

### Backend
cd backend  
pip install -r requirements.txt  
python app.py  

### Mobile app
cd mobile-app  
npm install  
npx expo start
